Portfolio site using GatsbyJS and Wordpress REST API.

1. **Install dependencies.**

   ```shell
   npm install
   ```

3. **Start the site in `develop` mode.**   

   ```sh
   gatsby develop
   # Running at http://localhost:8000
   ```

3. **Start the site in `build` mode.** 

   ```sh
   gatsby build
   ```
